Quake

This is a port of Quake to android 

In order to compile and run  Quake on the emulator and/or an Android device, you must:

	1) install android ndk and sdk
    2) run ./debug.sh , will compile all files and install on you phone
	6) put data.zip on you phone /sdcard/quake/
	7) run the game
